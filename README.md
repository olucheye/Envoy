# Envoy
__NodeJS Shopping Assistant - Implementing CRUD app &amp; PassportJS Authentication__

![Web App Screenshot](https://i.imgur.com/qSDMLUO.png)

## Project goal

- User Authentication With Passport.js
- Create User Account
- Update User Account with wishlist
- Authenticate on log-in
- Logout

## Clone & Build on local

Clone the repository, install node packages and verify routes locally

```
//on local
git clone URL
cd myproject
npm install
npm start dev

```
